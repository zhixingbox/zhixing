# 知行盒子 - 互动式编程教育平台
## 森林守护者：小红帽的冒险之旅

[![Platform: GoPlus](https://img.shields.io/badge/Platform-GoPlus-blue.svg)](https://goplus.org)

## 项目简介

**知行盒子（ZhiXing Box）** 是一款创新的互动式编程教育产品，通过经典童话故事改编，将编程学习与沉浸式叙事体验完美结合。本项目基于 **GoPlus (SPX)** 编程语言开发，为青少年编程教育提供了一个富有创意和教育意义的学习场景。

### 核心故事线

本项目以"森林守护者"为主题，重新演绎了经典的小红帽故事：
- **小红帽**作为见习森林守护者，肩负着守护森林的使命
- **大灰狼**因被黑暗巫师控制而陷入危机
- 通过**勇气、智慧和合作**，主人公们最终建立联盟，共同对抗黑暗势力

## 技术特性

### 核心技术栈
- **编程语言**: GoPlus (SPX) - 专为创意编程设计的现代语言
- **开发平台**: GoPlus Builder - 可视化编程环境
- **资源管理**: 基于 JSON 的资源索引系统
- **版本控制**: Git - 支持团队协作开发

### 技术亮点

```go
// 示例：精灵动画与事件驱动编程
onBackdrop "moonlight_lake", => {
    setGraphicEffect ColorEffect, 30
    play "bgm", true
    
    // 角色互动逻辑
    wolf.say "快...快走...我控制不了自己...", 2
    redHood.say "也许...我可以帮你。", 2
    
    // 动画效果
    magicWater.glide 120, 10, 1.5
}
```

**实现功能**：
- ✅ 多场景切换与过渡效果
- ✅ 角色对话系统
- ✅ 精灵移动与动画控制
- ✅ 音效与背景音乐同步
- ✅ 图形特效（颜色、透明度、闪烁等）
- ✅ 事件广播与监听机制
- ✅ 复杂的剧情逻辑编排

## 项目结构

```
zhixing/
├── main.spx                 # 主场景控制逻辑
├── redHood.spx / redHood1.spx  # 小红帽角色脚本
├── wolf.spx / wolf1.spx     # 大灰狼角色脚本
├── magicDeer.spx            # 神鹿角色
├── magicWater.spx           # 魔法道具
├── cureHerb.spx             # 治疗草药
├── grandma1.spx             # 奶奶（森林守护者）
├── [其他角色].spx           # 森林动物角色
└── assets/
    ├── backdrops/           # 场景背景（6+ 个场景）
    ├── sprites/             # 角色精灵素材（15+ 个角色）
    ├── sounds/              # 音效库
    └── index.json           # 资源索引
```

## 教育价值

### 编程概念学习
- **变量与函数**: 自定义函数（如 `播放背景序列()`、`dance()`）
- **事件驱动**: `onStart`、`onBackdrop`、`onMsg` 等事件处理
- **循环与条件**: `repeat`、`wait` 等流程控制
- **多媒体编程**: 音效、动画、图形效果的综合运用

### 软技能培养
- **创意思维**: 通过故事创作培养想象力
- **逻辑思维**: 剧情逻辑的编排需要严谨的因果推理
- **问题解决**: 调试代码、优化效果的过程培养解决问题的能力
- **项目管理**: 多文件协作、资源管理的实践经验

## 快速开始

### 环境要求
- GoPlus 开发环境
- GoPlus Builder（推荐）

### 运行项目
1. 克隆仓库
```bash
git clone https://github.com/zhixingbox/zhixing.git
cd zhixing
```

2. 使用 GoPlus Builder 打开项目

3. 运行 `main.spx` 开始体验完整故事

### 自定义开发
- 修改 `.spx` 文件可调整角色行为和对话
- 替换 `assets/` 中的资源可更换视觉风格
- 添加新的 `.spx` 文件可扩展新角色
